A lightweight, gentle, and customizable desktop floating widget.

YuClock supports **Desktop Widget Mode** and **Timer Clock Mode**: you can place important goals, to-dos, and progress on your desktop, or track time for studying, coding, reading, and resting — generating a daily timeline and statistics.

## Features

- **Desktop Widget Mode**: Countdowns, to-do tasks, progress bars
- **Timer Clock Mode**: Tag-based timing, pause/resume, end recording
- **Mini Clock**: Minimized view retaining core timer info — ideal for floating during focus sessions
- **Daily Timeline**: Records where each segment of time was spent
- **Daily Statistics**: Time and percentage breakdown by tag
- **Global Customization**: Theme color, transparency, font, common tags, and more
- **System Tray Control**: Show/hide, settings, mode switching, and quit

## Two Modes

### Desktop Widget

Ideal for keeping on your desktop to check at a glance:

- How long until a target date
- Today's or upcoming to-dos
- Current goal progress

### Timer Clock

Ideal for studying, research, coding, or reading:

- Select a common tag
- Start timing
- Pause or resume
- End and write to timeline
- Auto-generate daily statistics

---

## Customization

YuClock lets you match your desktop style with adjustable appearance and behavior:

- Global theme color
- Transparency
- Countdown font
- Mute toggle
- Launch at startup
- Common timer tags
